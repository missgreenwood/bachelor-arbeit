# Struktur

Ziel: Match Strommessung und Zeitdauer/Anzahl aktiver RPis 


1. Alle RPis powered HPL: Plotfiles STROM1a und STROM1b
	a.  x: Zeit
		y: Stromverbrauch in W
		
2. Nur aktive RPis powered HPL: Plotfiles STROM2a und STROM2b
	a.  x: Zeit
		y: Stromverbrauch in W

3. Alle RPis powered STREAM: Plotfiles STROM3a und STROM3b
	a.  x: Zeit
		y: Stromverbrauch in W

4. Nur aktive RPis powered HPLinpack: Plotfiles STROM4a und STROM4b
	a.  x: Zeit
		y: Stromverbrauch in W

5. Messreihe 1 und Messreihe 2 HPLinpack: Plotfile STROM5

6. Messreihe 1 und Messreihe 2 STREAM: Plotfile STROM6


## Inputfiles erstellen 

1. Gnuplot/energymatch1.txt

2. Gnuplot/energymatch2.txt

3. Gnuplot/energymatch3.txt

4. Gnuplot/energymatch4.txt



## Gnuplot-Skripte f√ºr 1-4 (Plotfiles 'STROM1' - 'STROM4')

# STROM1a

set terminal postscript eps noenhanced defaultplex leveldefault color colortext
set ylabel "W"
# set yrange [70:120]
unset xtics
set xlabel "t"
set title "Stromverbrauch HPL, Messreihe 1"
set output 'strom1a.eps'
set style data lines
plot 'energymatch1.txt' using 0:1 title 'Stromverbrauch'
